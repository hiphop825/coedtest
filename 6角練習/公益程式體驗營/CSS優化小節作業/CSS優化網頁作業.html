下列的emmet語法，哪一項是錯的？

  • ! = HTML5環境
  • link:src = 匯入CSS
  • p*4 = 顯示4個p段落
  • 1 = 展開h1的開頭與結尾HTML標籤
  • lorem = 自動就幫我們寫完網頁
